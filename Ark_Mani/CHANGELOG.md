# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-02-02

### Added
- Initial release of Portfolio Risk Assessment Tool
- Streamlit-based web interface with five main pages
  - Portfolio Input (CSV upload, manual entry, samples)
  - Risk Analysis dashboard
  - Portfolio Optimization interface
  - Scenario Testing with LLM integration
  - Report Generation and Export
- FastAPI backend with RESTful API
- Risk calculation service implementing:
  - Value at Risk (VaR) at 95% and 99% confidence levels
  - Conditional VaR (Expected Shortfall)
  - Sharpe Ratio calculation
  - Sortino Ratio for downside risk
  - Maximum Drawdown analysis
  - Beta coefficient calculation
  - Skewness and Kurtosis metrics
- Monte Carlo simulation engine
  - Support for 1,000 to 100,000 simulation paths
  - Configurable time horizons (1 day to 1 year)
  - Correlation-preserving simulations
  - Confidence interval calculation
- Portfolio optimization using CVXPY
  - Minimum variance optimization
  - Maximum Sharpe ratio optimization
  - Risk parity allocation
  - Efficient frontier generation
  - Custom constraint support
- Market data integration via yFinance
  - Historical price data fetching
  - Real-time current price retrieval
  - Asset information lookup
  - Caching layer for performance
- LLM service integration with Hugging Face
  - Natural language scenario interpretation
  - Structured output generation
  - Support for GPT-J and BLOOM models
- Kubernetes deployment configurations
  - Frontend deployment with LoadBalancer service
  - Backend deployment with Horizontal Pod Autoscaler (2-10 replicas)
  - LLM service deployment with GPU support
  - PostgreSQL StatefulSet for data persistence
  - Redis deployment for caching
  - Parallel job execution for Monte Carlo simulations
  - CronJob for scheduled portfolio analysis
- Docker containerization
  - Multi-stage builds for optimized images
  - Docker Compose for local development
  - Health checks for all services
- Monitoring and observability
  - Prometheus metrics endpoints
  - Grafana dashboard configurations
  - Health and readiness probes
  - Structured logging
- Comprehensive documentation
  - Main README with architecture and features
  - Quick Start guide for rapid deployment
  - Complete technical documentation
  - Presentation materials (25 slides)
  - API documentation (auto-generated by FastAPI)
  - Contributing guidelines
- Security features
  - Input validation with Pydantic
  - SQL injection prevention via SQLAlchemy ORM
  - Rate limiting support
  - Environment variable management
  - CORS configuration

### Technical Specifications
- Python 3.11+ support
- Support for 1000+ assets in portfolio
- Handles 100+ concurrent users with autoscaling
- Sub-5-second analysis for typical portfolios
- 99.9% uptime target with Kubernetes
